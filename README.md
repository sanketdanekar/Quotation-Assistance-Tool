# Quotation Assistance Tool

## Project Overview
Quotation Assistance Tool is an SAP ABAP application developed to automate and simplify quotation management.

## Technologies Used
- SAP ABAP
- SAP SD
- ALV Reports
- VBAK
- VBAP

## Features
- Customer quotation management
- Sales inquiry analysis
- ALV report display
- Sorting and subtotal calculations
- Interactive reporting

## SAP Tables Used
- VBAK (Sales Document Header)
- VBAP (Sales Document Item)

## Project Outcome
The project reduces manual effort, improves quotation processing speed, and provides better visibility into sales data.
